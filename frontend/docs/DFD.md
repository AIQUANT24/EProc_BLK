### Process Inventory

| Process | Name | Description |
|---------|------|------------|
| P1 | Supplier Management | Registration, KYC, identity management |
| P2 | Product Management | Product registration, versioning |
| P3 | BOM Management | BOM creation, import, versioning |
| P4 | DVA Calculation | Calculate domestic value addition |
| P5 | Compliance Engine | Classify suppliers, generate certificates |
| P6 | Fraud Detection | Rule-based and AI fraud detection |
| P7 | Blockchain Recording | Immutable compliance record storage |
| P8 | Verification API | External platform verification |
| P9 | Supply Chain Verification | Cross-reference origin claims |
| P10 | Analytics & Reporting | Dashboards, reports, policy simulation |
